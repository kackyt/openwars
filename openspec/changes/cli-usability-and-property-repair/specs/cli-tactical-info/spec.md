## ADDED Requirements

### Requirement: 地形情報の表示
カーソルがマップ上のタイルを指しているとき、その地形の種類と防御ボーナスを表示しなければならない (MUST)。
- 名称: `MasterDataRegistry` に準拠（例: 森、山）
- フォーマット: `[名称] (防: [数値])`

#### Scenario: 平地カーソル時の表示
- **GIVEN** カーソルが「平地」のマスにある
- **WHEN** 画面が描画されるとき
- **THEN** 情報パネルに `平地 (防: 1)` と表示されなければならない (SHALL)

### Requirement: 拠点耐久力の表示
カーソルが拠点（都市、工場、空港、港、首都）を指しているとき、現在の占領耐久力を表示しなければならない (MUST)。
- フォーマット: `占領: [現在値]/[最大値]`

#### Scenario: 都市カーソル時の表示
- **GIVEN** カーソルが占領耐久値120の「都市」にある（最大値200）
- **WHEN** 画面が描画されるとき
- **THEN** 情報パネルに `占領: 120/200` と表示されなければならない (SHALL)

### Requirement: ユニット状態の視覚化
行動済み、またはユニットを搭載している場合の視覚的な区別を提供しなければならない (MUST)。
- 行動済み: マップ上のシンボルを暗調（RATATUI DIM）で表示。
- 搭載中: シンボルの前に `*` を表示（例: `*h`。実表示では必要に応じて末尾スペースを含む）。

#### Scenario: 行動済みユニットの描画
- **GIVEN** アクティブプレイヤーの「軽戦車」がすでに「待機」済みである
- **WHEN** マップが描画されるとき
- **THEN** その戦車の記号（T）は `DIM` モディファイア（暗調）を伴って描画されなければならない (SHALL)

#### Scenario: 搭載中ユニットの描画
- **GIVEN** 「輸送ヘリ」の中に「歩兵」が１体搭載されている
- **WHEN** マップが描画されるとき
- **THEN** そのヘリの記号（h）の前に `*` が付与され、`*h`（実表示では必要に応じて末尾スペースを含む）と表示されなければならない (SHALL)
