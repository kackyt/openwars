# Specification: Allied Unit Passing

## Overview

自軍ユニットが占有するマスを、他の自軍ユニットが通過できるようにします。

## Requirements

### Requirement: 自軍ユニットの通過
- **WHEN** ユニットが移動経路を計算または実際に移動する際、その経路上のマスに自軍の別のユニットが存在する場合
- **THEN** そのマスを通過可能なものとして扱い、地形に応じた移動コストを支払って通り抜けることができる。

### Requirement: 移動終点の制限
- **WHEN** ユニットの移動の終点（目的地）に別の自軍ユニットが存在する場合
- **THEN** そのマスが積載可能な輸送ユニットであり、かつ移動ユニットが積載条件を満たしている場合を除き、そのマスに留まることはできない。

### Requirement: 敵軍ユニットの遮断
- **WHEN** ユニットの移動経路上に敵軍のユニットが存在する場合
- **THEN** そのマスを通過することはできず、経路探索において「壁」として扱われる。

### Requirement: 敵軍ユニットによる ZOC (Zone of Control) の適用
- **WHEN** ユニットが移動中に、敵軍ユニットに隣接するマス（ZOC）に進入した場合
- **THEN** そのマスに自軍ユニットが存在するかどうかにかかわらず、進入した時点で移動を終了しなければならない（強制停止）。
- **REASON** 自軍ユニットによる「通過許可」は「敵軍の妨害（ZOC）」を無効化するものではない。
