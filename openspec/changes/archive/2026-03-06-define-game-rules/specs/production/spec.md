# production Specification Delta

## ADDED Requirements

### Requirement: ユニットの生産 (Unit Production)
システムは、各プレイヤーが自国の特定の拠点において、自身の資金を消費して新しいユニットをマップ上に配置（生産）する機能をサポートしなければならない (SHALL)。

#### Scenario: 生産条件のバリデーション（成功）
- **WHEN** プレイヤーがユニットの生産を試み、その場所が「自国の都市または空港」であり、かつ「自国の首都から3マス以内」に存在し、さらにプレイヤーが「ユニットの生産コスト以上の資金」を保有しているとき
- **THEN** 生産の実行を許可し、プレイヤーの資金から生産コストを減算した上で、指定されたユニットをその座標に配置する

#### Scenario: 生産条件のバリデーション（失敗 - 距離制限）
- **WHEN** プレイヤーがユニットの生産を試みた場所が自国の拠点であっても、「自国の首都から4マス以上離れている」とき
- **THEN** 生産の実行を拒否し、資金の減算やユニットの配置を行わない

#### Scenario: 生産条件のバリデーション（失敗 - 資金不足）
- **WHEN** プレイヤーがユニットの生産を試みたが、ユニットの生産コストに対してプレイヤーの現在の資金が不足しているとき
- **THEN** 生産の実行を拒否し、資金の減算やユニットの配置を行わない
