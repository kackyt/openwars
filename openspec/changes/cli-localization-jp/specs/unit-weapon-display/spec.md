## ADDED Requirements

### Requirement: 武器名の保持
MUST: ユニットは、マスターデータで定義された武器名をコンポーネントとして保持しなければならない。

#### Scenario: ユニット生成時の武器名設定
- **WHEN** ユニットが生産または初期配置されたとき
- **THEN** マスターデータの「武器1」「武器2」に定義された名称が、当該ユニットのコンポーネントに保存されること。

### Requirement: 武器名の表示
MUST: CLIは、ユニットの詳細情報を表示する際、汎用的なラベル（Ammo 1/2）ではなく、個別の武器名を表示しなければならない。

#### Scenario: 武器名付き情報パネルの表示
- **WHEN** ユーザーが武器を持つユニットにカーソルを合わせたとき
- **THEN** インスペクション・パネルに「戦車砲S: 9/9」のように武器名と残弾数が表示されること。
