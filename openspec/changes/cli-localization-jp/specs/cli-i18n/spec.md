## ADDED Requirements

### Requirement: CLI UIの日本語化
MUST: システムは、CLIのすべての表示要素（メニュー、ラベル、ヘルプ、ログ）を日本語で提供しなければならない。

#### Scenario: マップ選択画面の表示
- **WHEN** ユーザーがマップ選択画面を表示したとき
- **THEN** タイトル、マップリストのラベル、操作ヘルプが日本語で表示されること。

#### Scenario: ゲーム内全般のヘルプ表示
- **WHEN** ユーザーがゲーム中に操作説明パネルを確認したとき
- **THEN** 「移動」「アクション」「キャンセル」などの操作説明が日本語で表示されること。

### Requirement: ゲーム内ログの日本語化
MUST: システムは、ゲーム内での重要なアクションや結果を日本語のログメッセージとして記録しなければならない。

#### Scenario: ユニット選択時のログ
- **WHEN** ユーザーがマップ上のユニットを選択したとき
- **THEN** 「(座標) のユニットを選択しました」といった日本語のログが出力されること。

#### Scenario: ターンの終了ログ
- **WHEN** ユーザーが「ターン終了」を選択したとき
- **THEN** 「ターンを終了しました」といった日本語のログが出力されること。
